# dotfiles

Gautam Nair (g23polar) dotfiles.

## To install
 From here, on any new machine it's just:

 ```bash                          
   git clone <repo> ~/dotfiles                                                                                                                                                                                             
   bash ~/dotfiles/install.sh                                                                                                                                                                                              ```                                                                                                                                                                                                                       


## Quick Start

```bash
# install / run
```
